# Buda Client

Buda.com api client

## Installation

```bash
$ gem install budaClient
```

Or add to your Gemfile:

```ruby
gem "budaClient"
```

```bash
bundle install
```

## Testing

To run the specs you need to execute, **in the root path of the gem**, the following command:

